# 📊 Personal Tracker Dashboard (Excel + Python)

## 📌 Overview

The **Personal Tracker Dashboard** is a data-driven project that uses Excel as a data source and Python for analysis and visualization.
It helps users track their daily activities such as expenses, study hours, and productivity, and transforms raw data into meaningful insights through charts and summaries.

This project demonstrates practical use of data analysis using Python and Excel integration, making it useful for students and beginners in data science.

---

## 🎯 Objectives

* To analyze personal daily data efficiently
* To generate insights from Excel datasets
* To visualize data using graphs and charts
* To build a simple and interactive tracking dashboard

---

## 🚀 Features

* 📥 Import data from Excel files
* 📊 Automatic calculations (totals, averages, categories)
* 📈 Visual dashboards (pie charts, bar graphs)
* 🧠 Insight generation (highest spending, productivity trends)
* 💾 Export results back to Excel (optional)

---

## 🛠️ Technologies Used

* **Python**
* **Pandas** – data manipulation
* **Matplotlib** – data visualization
* **OpenPyXL** – Excel file handling
* **Microsoft Excel** – data storage

## 📊 Dataset Description

The Excel file (`tracker.xlsx`) contains the following columns:

| Date       | Category | Amount | Hours | Notes        |
| ---------- | -------- | ------ | ----- | ------------ |
| 01-04-2026 | Food     | 200    | 2     | Lunch        |
| 02-04-2026 | Study    | 0      | 5     | DSA practice |

* **Category** → Expense or activity type
* **Amount** → Money spent
* **Hours** → Time spent on activity
